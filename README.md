## 📌 О проекте

**Awesome Kanban Board** — веб-приложение для управления задачами с интерактивной канбан-доской. Позволяет организовать рабочий процесс через четыре статуса: **Backlog**, **Ready**, **In Progress**, **Finished**.

**Ключевые возможности:**
- Drag-and-drop задач между колонками
- Назначение приоритетов
- Авторизация через email (Firebase Auth)
- Автосохранение в облачной базе данных
