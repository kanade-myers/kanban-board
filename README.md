## 📖 Описание проекта
Это учебный фронтенд-интерфейс, имитирующий работу с системой проверки контрагентов "СКАН". Проект включает в себя сложную форму с фильтрами, клиентскую валидацию ИНН и интеграцию со сторонним API для получения данных о компании.

## 🚀 Функционал
- **Валидация ИНН:** Проверка корректности ввода (длина, контрольные цифры) в реальном времени.
- **Форма с фильтрами:** Настраиваемые параметры поиска компаний.
- **Интеграция с API:** Отправка запроса на [Название API, например: `https://api-scan.ru/v1/search`] и отображение полученных данных.
- **Отображение результатов:** Таблица/Карточки с информацией о компании.

## 🛠️ Стек технологий
- **HTML5:** Семантическая структура.
- **CSS3:** (Flexbox/Grid, Адаптивная верстка).
- **JavaScript (ES6+):** Работа с DOM, `fetch` для запросов, кастомная валидация.
